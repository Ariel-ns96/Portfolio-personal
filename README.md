# Portfolio Personal

Este es mi portfolio personal, desarrollado con **Next.js**, **React** y **Tailwind CSS**, pensado para mostrar mis proyectos, trabajos y habilidades de manera interactiva y dinámica.  

Este portfolio permite explorar mis proyectos y experiencias, y experimentar con la aplicación de manera interactiva, agregando, editando o borrando datos para apreciar cómo funciona (Los cambios son solo locales y no afectan la información real).

## Tecnologías

- **Frontend:** React, Next.js, Tailwind CSS, Material UI  
- **Internacionalización:** i18next  
- **Otras herramientas:** Vercel para despliegue  

---

## Características principales

- Interfaz moderna y responsiva.  
- Soporte para **modo claro/oscuro**.  
- Secciones dinámicas de proyectos y contactos.  
- Formularios para agregar y editar información directamente desde la interfaz.  
- Sección de redes sociales con enlaces y correo electrónico.  
- Galería de trabajos y proyectos interactiva.  
- Soporte multilenguaje (español e inglés).
